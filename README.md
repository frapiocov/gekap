TSW (Software Technology for Web) project developed with **Java**, **JavaScript**, **HTML**, and **CSS**.

## 🚀 Technologies Used
- 🔹 **Backend**: Java (Spring Boot)
- 🔹 **Frontend**: JavaScript, HTML, CSS
- 🔹 **Database**: MySQL / PostgreSQL
- 🔹 **Local Server**: Adobe
- 🔹 **Version Control**: Git

## ✨ Features
✅ User registration and authentication 🔐  
✅ Product management (add, edit, delete) 📦  
✅ Shopping cart and checkout 🛍️  
✅ Admin dashboard 📊  

## 🛠️ Installation

### 📌 Prerequisites
- ☕ Java JDK 17+
- 🗄️ MySQL
- ⚙️ Maven
- 🌐 Server for local environment

### 🔧 Instructions
1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/ecommerce.git
   cd ecommerce
   ```
2. **Configure the database** 📂
   - Create a database and update the `application.properties` file with the correct credentials.
3. **Start the backend** 🚀
   ```bash
   mvn spring-boot:run
   ```
4. **Start the frontend** 🎨
   ```bash
   cd frontend
   npm install
   npm start
   ```
